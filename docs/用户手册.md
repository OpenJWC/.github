## 快速入门

### 什么是OpenJWC？

OpenJWC是一个私有化部署的教务资讯系统，为您提供：

- **实时通知**: 第一时间获取教务信息
- **AI问答**: 智能问答系统，快速获取答案
- **现代界面**: 美观易用的用户界面
- **隐私安全**: 数据安全，隐私保护

### 支持的平台

- **Android客户端**: 支持Android 8.0及以上版本
- **Web管理面板**: 支持现代浏览器（Chrome、Firefox、Edge等）
- **QQ机器人**: 支持OneBot协议的QQ机器人框架

## Android客户端使用指南

### 安装应用

1. 访问[OpenJWCClient Releases](https://github.com/OpenJWC/OpenJWCClient/releases)
2. 下载最新版本的APK文件
3. 在手机上安装APK文件

### 首次设置

**启动应用后，您需要配置服务器地址：**

```
1. 打开应用
2. 进入"设置" -> "网络配置"
3. 输入服务器地址: http://your-server-ip:8000
4. 点击"保存"
```

### 主界面介绍

#### 底部导航栏

- **聊天**: 与AI助手对话
- **资讯**: 浏览最新的教务通知
- **我**: 个人设置和账户管理

#### 首页功能

**通知列表：**

- 显示最新的教务通知
- 支持下拉刷新
- 点击查看通知详情

**通知卡片：**

- 标题：通知标题
- 摘要：通知内容摘要
- 时间：发布时间
- 标签：分类标签（如：教务信息、实践教学）

### 功能详解

#### 1. 浏览通知

**查看通知列表：**

1. 在查看通知列表
2. 下拉刷新获取最新通知
3. 点击通知查看详情

**通知详情页面：**

- 完整的通知内容
- 相关附件下载
- 添加至附件功能

#### 2. AI助手

**开始对话：**

1. 点击底部"聊天"标签
2. 在输入框输入您的问题
3. 点击发送按钮
4. 查看AI回复

**支持的对话类型：**

- **信息查询**："下周一有哪些考试？"
- **时间查询**："期末考试什么时候开始？"
- **地点查询**："计算机学院的考试在哪里？"
- **相关问题**："关于补考的通知有哪些？"

#### 3. 个性化设置

**主题设置：**

- 浅色主题
- 深色主题
- 跟随系统

**服务器设置：**

- 服务器地址配置
- API密钥管理

## QQ机器人使用指南

### 配置机器人

#### 1. 安装OneBot实现

**安装NapCat（推荐）：**

1. 下载NapCat
2. 安装并配置NapCat
3. 启动NapCat服务

#### 2. 配置机器人连接

**编辑配置文件：**

```bash
# 编辑.env文件
ONEBOT_ACCESS_TOKEN=your-access-token
ONEBOT_WS_URLS=ws://127.0.0.1:3001

OPENJWC_AUTH_KEY=your-auth-key
OPENJWC_HOST=http://localhost:8000
OPENJWC_PORT=8000

OPENJWC_NOTICE_GROUPS=123456789,987654321
OPENJWC_CHECK_INTERVAL=5
```

#### 3. 启动机器人

```bash
# 启动机器人
python bot.py
```

### 使用命令

#### 基础命令

**通知相关：**

- `/通知` - 获取最新通知
- `/通知 搜索 关键词` - 搜索通知
- `/通知 分类 分类名` - 按分类查看通知

**帮助命令：**

- `/帮助` - 显示帮助信息
- `/关于` - 显示机器人信息

#### 高级命令

**管理员命令：**

- `/admin 用户列表` - 查看用户列表
- `/admin 系统状态` - 查看系统状态
- `/admin 重启爬虫` - 重启爬虫服务

### 功能使用

#### 1. 自动推送

**配置推送群组：**

1. 在.env文件中设置`OPENJWC_NOTICE_GROUPS`
2. 输入要推送的QQ群号（多个群用逗号分隔）
3. 重启机器人

**推送内容：**

- 新通知标题
- 通知摘要
- 链接（点击可查看详情）
- @全体成员提醒

#### 2. 互动查询

**查询通知：**

1. 在QQ群中输入命令
2. 机器人会回复查询结果
3. 支持多轮对话

#### 3. 自定义回复

**设置自定义回复：**

1. 在Web管理面板的"QQ机器人设置"中
2. 添加自定义回复规则
3. 设置触发词和回复内容

#### 2. 群组管理

**多群组支持：**

- 可配置多个QQ群
- 每个群可单独设置推送规则
- 支持群组白名单

### 个性化定制

**Android客户端：**

- 浅色/深色主题
- 自定义主题颜色
- 自定义背景图片
